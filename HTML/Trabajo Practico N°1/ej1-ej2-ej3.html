Programacion Web

La programación web es el proceso de creación, diseño y mantenimiento de sitios y aplicaciones que se ejecutan en navegadores de internet. Combina lógica y lenguajes de programación para construir experiencias interactivas, funcionales y seguras, cubriendo tanto la parte visual como la lógica del servidor.

Componentes Principales:Frontend (Lado del cliente): Es lo que el usuario ve e interactúa, creado con HTML (estructura), CSS (diseño) y JavaScript (interactividad).Backend (Lado del servidor): Es la lógica detrás de escena, manejando bases de datos, usuarios y servidores. Se utilizan lenguajes como PHP, Python, Ruby o Node.js.

HTML

HTML (HyperText Markup Language o Lenguaje de Marcado de Hipertexto) es el estándar básico para estructurar y desplegar el contenido de páginas web. Utiliza un sistema de etiquetas (tags) para organizar elementos como textos, títulos, imágenes, enlaces y multimedia, creando el esqueleto estructural que los navegadores interpretan.
Características clave de HTML:Lenguaje de Marcado, no de Programación: Define la estructura y el formato, no la lógica de programación.Basado en Etiquetas (tags): Utiliza elementos encerrados en ángulos, por ejemplo, h1 para un título o p para un párrafo.Estructura fundamental: Todo documento HTML suele dividirse en un encabezado (head - metadatos) y un cuerpo (body - contenido visible).Hipertexto: Permite enlazar diferentes páginas o sitios web entre sí a través de hipervínculos.
